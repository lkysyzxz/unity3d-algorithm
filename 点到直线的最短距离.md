## 点到线段的最短距离
&emsp;&emsp;根据点到直线的距离的计算方式，把线段看成是直线，最后计算出t，然后根据t的值将t缩小到[0,1]之间。

```
if t < 0
than t = 0
if t > 1
than t = 1
```
&emsp;&emsp;这样就可以计算点到线段的最短距离了。
